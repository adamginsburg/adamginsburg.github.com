Converting CCDSoft FIT to fits files


  1. Download FITtofits.cl and place it in the directory where you have stored your images
  2. Open IRAF with the command pyraf and changed to the directory with your images
  3. Run the command cl < FITtofits.cl
  4. Check that it worked:
    1. pick an image (the image name before the .xxx extension is [imname] below)
    2. run imstat [imname].FIT
    3. compare to imstat [imname].fits
    The first command should give an error, the second command will give you sensible results.
