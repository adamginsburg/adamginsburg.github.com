Interests - mostly acronyms

An explanation of acronyms (as seen in my facebook profile or heard in confusing conversation):
pAGBs - Post-Asymptotic Giant Branch objects, sometimes also Pre-Planetary or Protoplanetary Nebulae
LBVs - Luminous Blue Variables
awk - the gnu awk program
sed - unix stream editor
mountains - I guess they're not acronyms
perl - Practical Extraction and Report Language
bash - the unix Bourne Again Shell
cgi - common gateway interface, i.e. perl scripts on the web
pdl - the perl data language, a free (but incomplete) alternative to idl
regex - regular expressions
sql - structured query language
stellar death - what do you think? I think: awesome. e.g. Eta Carinae
2dust - radiative transfer model by Toshiya Ueta
php - PHP: Hypertext Preprocessor
html - hypertext markup language
vim - shortest unambiguous abbreviation for "visual"... improved
co - Colorado
nicfps - Near Infrared Camera/Fabry Perot Spectrometer
python - whitespace-dependent programming language
pyraf - python-based IRAF
idl - interactive data language
iraf - Image Reduction and Analysis Facility
collisions - when two things crash into each other
asm - assembly language (e.g. asm56300, Motorola assembly)
qm - quantum mechanics
57005 - 0xDEAD
etalons - Fabry-Perot spectrometers
